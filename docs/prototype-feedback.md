# 原型系统反馈文档

## 原型系统说明

#### 1.1 技术框架

- 前端框架：Vue3 + TypeScript
- UI 组件库：Element Plus
- 状态管理：Pinia
- 路由管理：Vue Router
- HTTP 客户端：Axios

#### 1.2 页面组件

- 登录页面
- 主布局
- 首页仪表盘
- 游戏节点管理
- 游戏平台管理
- 游戏卡片管理
- 游戏实例管理

## 设计规范总结

### 1. 布局规范

- 页面采用卡片式布局
- 表格展示数据列表
- 分页控制在底部
- 操作按钮统一位置

### 2. 交互规范

- 表单验证即时反馈
- 操作结果即时提示
- 状态变更即时更新
- 加载状态明确显示

### 3. 视觉规范

- 使用 Element Plus 默认主题
- 状态标签颜色统一
- 按钮样式统一
- 表格样式统一
